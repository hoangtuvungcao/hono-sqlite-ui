# sqlite-ui-monorepo

This repository contains the SQLite Administration UI Dashboard implementations for both JavaScript/TypeScript (Hono) and Python (FastAPI).

*Kho lưu trữ này chứa hai phiên bản giao diện quản trị cơ sở dữ liệu SQLite dành cho JavaScript/TypeScript (Hono) và Python (FastAPI).*

---

## Directory Structure (Cấu trúc thư mục)

```text
test_pk_npm/
├── hono-sqlite-ui/         # JavaScript/TypeScript package for Hono (NPM)
└── fastapi-sqlite-ui/      # Python package for FastAPI (PyPI)
```

---

## 1. hono-sqlite-ui (JavaScript/TypeScript)

### English
A lightweight, drop-in admin panel UI for SQLite database management, built specifically for Hono routers running on Bun or Node.js.
- **NPM Package**: [hono-sqlite-ui](https://www.npmjs.com/package/hono-sqlite-ui)
- **Framework**: Hono (Node.js / Bun)
- **Read more**: [hono-sqlite-ui README](hono-sqlite-ui/README.md)

### Tiếng Việt
Giao diện quản lý SQLite Admin Panel gọn nhẹ, cắm-là-chạy được thiết kế dành riêng cho các router Hono chạy trên môi trường Bun hoặc Node.js.
- **NPM Package**: [hono-sqlite-ui](https://www.npmjs.com/package/hono-sqlite-ui)
- **Framework**: Hono (Node.js / Bun)
- **Xem thêm**: [hono-sqlite-ui README (tiếng Việt)](hono-sqlite-ui/README.md)
